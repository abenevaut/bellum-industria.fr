# Mono C# on debian

http://mono-project.com/DistroPackages/Debian
apt-get install mono-complete

utiliser l'ide monodevelop


Pour apache2
https://help.ubuntu.com/community/ModMono
sudo apt-get install libapache2-mod-mono
sudo apt-get install libapache2-mod-mono mono-apache-server mono-devel mono-runtime
sudo apt-get install mono-apache-server2

sudo a2enmod mod_mono
(si mod_mono n'existe pas refaire sudo apt-get install libapache2-mod-mono)

a2dismod mod_mono
a2enmod mod_mono_auto



sudo /etc/init.d/apache2 restart
