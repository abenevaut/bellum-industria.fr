# MAMP


Se connecter au server mysql en ligne de commande :

	/Applications/MAMP/Library/bin/mysql --host=localhost -uroot -p
	/Applications/MAMP/Library/bin/mysql --host=localhost -uroot -p TABLE_NAME < /tmp/dump.sql


MySQL Dump

	/Applications/MAMP/Library/bin/mysqldump -u root -p --databases DATABASE_NAME > DUMP_FILENAME.sql

MySQL Dump sans donnee

	/Applications/MAMP/Library/bin/mysqldump -u root --no-data -p --databases DATABASE_NAME > DUMP_FILENAME.sql

Socket

  /Applications/MAMP/tmp/mysql/mysql.sock
